# Athens Analysis - Análisis Estructural

## Descripción 📑
Esta aplicación permite realizar análisis estructurales de vigas de concreto de manera sencilla e intuitiva. Está diseñada para ingenieros y estudiantes de ingeniería civil que deseen evaluar el rendimiento y la seguridad de las vigas en diferentes condiciones de carga.

## Características
- **Entradas Personalizables**: Permite a los usuarios ingresar dimensiones, tipo de concreto y cargas aplicadas.
- **Cálculos Estructurales**: Realiza cálculos para determinar el momento flector de la viga.
- **Comparación de Materiales**: Permite comparar diferentes tipos de concreto y sus propiedades.
- **Exportación de Resultados**: Opción para exportar resultados en formato PDF o CSV.

## Tecnologías Usadas
- **React**: Biblioteca de JavaScript para construir interfaces de usuario.
- **TypeScript**: Lenguaje de programación que añade tipos estáticos a JavaScript.
